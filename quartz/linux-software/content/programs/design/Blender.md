- URL: https://www.blender.org/
### Install

```shell
tar xvf blender-5.1.0-linux-x64.tar.xz
```

### blender.desktop

```shell
# copy sample desktop file to user directory
cp blender-5.1.0-linux-x64/blender.desktop ~/.local/share/applications/
# edit Exec, Icon path in desktop file
nano ~/.local/share/applications/blender.desktop
```
